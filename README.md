## Setup

First, install [`uv`](https://docs.astral.sh/uv/).

Download here: https://docs.astral.sh/uv/getting-started/installation/

Then, create a `.venv` with:

```bash
uv sync
```

You can add new packages like this:

```bash
uv add <package_name>
```
